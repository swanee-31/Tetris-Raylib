# Tetris RAYLib
Yes, another Tetris game...

This code is for educational purpose : learning Raylib and exposing usage of some coding usages (patterns, polymorphism...).

This project is using CMake.

To build it, you need to install CMake and type :

cmake -S . -B build

At 1st time, it can take some minutes because it will download the Raylib libraries.

Then:
cmake --build build

The build/debug version of the executable can be found there:
./build/Debug/Tetris-Raylib

Enjoy
